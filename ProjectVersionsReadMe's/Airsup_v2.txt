Airsup_v2.ts Explanation
========================

Purpose
- Keeps v1 gameplay logic and adds first custom HUD layer.

Main Additions Over v1
- Per-player HUD creation with root container and score widgets.
- Static left/right bar backplates (visual only).
- HUD creation on mode start for existing players and on player join.

Functions Added/Expanded
- createHUD(player): builds HUD once per player using unique names from player object ID.

What Remains Same as v1
- Capture-point scoring model and scoreboard stat events.
- Player stat variable model and row updates.

Key mod API Additions
- AddUIContainer, AddUIText, GetUIRoot, FindUIWidgetWithName.
- UI layout/style types: UIAnchor, UIBgFill, CreateVector.
